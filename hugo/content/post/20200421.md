ブランチ`v1`〜`v2`間の各ファイルの追加・削除行数の合計を表示し、追加行数降順に並べる
```bash
git log --numstat --no-merges --pretty="" v1 v2 | grep -v "dist/" | grep -E ".+\.(ts|tsx)" | sort -nr
```
